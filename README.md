# 💬 ProChat - Professional Real-Time Messaging System

ProChat is a modern, professional-grade peer-to-peer messaging application built with **Laravel 12** and **Tailwind CSS**. It features a clean UI, read receipts, and a streamlined user experience for seamless communication.

---

## ✨ Features

- **User Authentication:** Secure login and registration.
- **Real-time Feel:** Optimized chat interface for smooth conversations.
- **Read Receipts:** Track message status with `is_read` and `read_at` timestamps.
- **Modern UI:** Responsive design using Tailwind CSS and Instrument Sans font.
- **Secure:** Built-in protection against common web vulnerabilities.

## 🛠️ Tech Stack

- **Framework:** Laravel 12
- **Frontend:** Tailwind CSS v4, Blade Templates
- **Database:** MySQL / PostgreSQL
- **Icons:** FontAwesome 6

---

## 🚀 Getting Started

Follow these steps to get a local copy up and running.

### 1. Prerequisites

- PHP >= 8.2
- Composer
- Node.js & NPM
- MySQL or any supported database

### 2. Installation Steps

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/ChamodyaGit/CP-Chat-App.git
    cd CP-Chat-App
    ```

2.  **Install PHP dependencies:**

    ```bash
    composer install
    ```

3.  **Install Frontend dependencies:**

        ```bash
        npm install && npm run build
        ```

4.  **Configure Environment & Database:**

# Create .env file from example

```bash
cp .env.example .env
```

# Generate application key

```bash
php artisan key:generate
```

# Run database migrations

```bash
php artisan migrate
```
